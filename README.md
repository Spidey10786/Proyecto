# 📊 Dashboard Administrativo

## Descripción
Este proyecto es un dashboard administrativo desarrollado con HTML5 y CSS3, aplicando CSS Grid para la estructura principal y Flexbox para componentes internos. Simula un sistema de gestión de usuarios, ventas e inventario.

---

## Tecnologías utilizadas
- HTML5 semántico
- CSS3 (Grid + Flexbox)
- Variables CSS
- Media Queries
- Diseño responsivo
- Principios básicos de accesibilidad

---

## Componentes principales
- Sidebar de navegación
- Header con buscador
- Tarjetas de estadísticas
- Tabla de datos
- Footer informativo

---

## Decisiones de diseño
Se utilizó CSS Grid para organizar toda la estructura del dashboard y Flexbox para alinear elementos internos como tarjetas y menú lateral. Se aplicaron efectos hover para mejorar la interacción del usuario.

---

## Accesibilidad
- Uso de etiquetas semánticas (`header`, `main`, `aside`, `footer`)
- Roles ARIA (`navigation`, `main`)
- Buen contraste de colores
- Navegación clara

---

## Responsividad
El dashboard se adapta a dispositivos móviles ocultando el sidebar y reorganizando el layout principal.
